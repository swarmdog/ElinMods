# Elin Fast Start Mod

Adds a **Fast Start** option to the new game mode selector. Skips the tutorial and Vernis questlines, granting all rewards, NPCs, and recipes immediately.

## Config

After first run, edit `BepInEx/config/mrmeagle.elin.faststart.cfg`:

```ini
[ExtraItems]
## Format: id:count,id:count,...
Items = money2:5,plat:20
```
